# Parent PDF Templates

Thư mục này chứa các mẫu PDF gửi phụ huynh:

- `parent-weekly-report.pdf`: Báo cáo học tập tuần.
- `parent-overdue-warning.pdf`: Cảnh báo bài trễ hạn.
- `parent-score-notice.pdf`: Phiếu thông báo điểm.
- `parent-consent-resource-policy.pdf`: Phiếu xác nhận/chính sách học liệu.

Các file `.md` là nội dung nguồn để chỉnh sửa lại nếu cần.
